# 15_本次调研新增遗漏 — 以人为中心的视频生成数据集补充

> 基于"训练视频生成模型（Video Diffusion/DiT）"的核心需求，本次调研识别出以下遗漏的高价值人类中心视频数据集。

---

## P0 级 — 强烈建议加入，与项目目标直接对齐

### OpenHumanVid (CVPR 2025, Fudan/Baidu)

| 维度 | 详情 |
|------|------|
| 规模 | 13.2M 高质量人像视频片段 |
| 标注 | 三类文本 prompt：human appearance caption、motion caption、combined caption |
| 重要性 | **专门为以人为中心的视频生成设计**，论文证明加入该数据集可显著提升生成人像视频的质量 |
| 发布年份 | 2024.11 (arXiv), CVPR 2025 |
| 访问等级 | B - 需发邮件申请 |
| 获取方式 | GitHub: <https://github.com/fudan-generative-vision/OpenHumanVid>，填写申请表后邮件获取下载链接 |
| 预计空间 | ~TB 级 |
| 关键模态 | RGB 视频、高质量多粒度文本描述 |
| 与项目关系 | 直接对齐"视频 Diffusion/DiT 生成模型以人为中心的视频反馈"需求 |

### EgoVid-5M (NeurIPS 2025)

| 维度 | 详情 |
|------|------|
| 规模 | 5M 第一人称视频片段 |
| 标注 | 细粒度动作标注、low-level 运动控制信息、ego-view action annotation |
| 重要性 | **专门为第一人称视频生成设计**，含精细动作控制标注，可直接用于条件视频生成训练 |
| 发布年份 | 2024.11 (NeurIPS 2025) |
| 访问等级 | A - 直接下载 |
| 获取方式 | <https://egovid.github.io/> |
| 预计空间 | ~TB 级 |
| 关键模态 | 第一人称 RGB 视频、动作标签、运动参数 |
| 与项目关系 | 第一人称交互视频生成的核心训练数据 |

### Action100M (Meta, 2026.01)

| 维度 | 详情 |
|------|------|
| 规模 | 147M 动作片段，14.6 年总时长，来源于 1.2M 教学视频 |
| 标注 | 全自动分层动作标注（hierarchical action labels），时间定位 |
| 重要性 | 目前最大的人类动作视频数据集之一，覆盖极广的人类操作场景 |
| 发布年份 | 2026.01 |
| 访问等级 | B - HF 可直接下载 10% preview，全量需申请 |
| 获取方式 | HuggingFace: `facebook/action100m-preview`，GitHub: <https://github.com/facebookresearch/Action100M> |
| 预计空间 | 全量 ~TB 级，preview ~百 GB 级 |
| 关键模态 | RGB 视频（YouTube URLs）、动作类别标签、时间边界 |
| 与项目关系 | 提供极大规模的人类动作监督信号，可用于预训练 |

## P1 级 — 建议考虑，重要人类行为数据源

### HowTo100M (2019)

| 维度 | 详情 |
|------|------|
| 规模 | 1.2M 教学视频，~136M 片段，约 15 年总时长 |
| 标注 | 自动语音识别 (ASR) 文本，视频-文本对 |
| 重要性 | 经典大规模视频-语言预训练数据集，大量人类演示和操作场景 |
| 发布年份 | 2019 |
| 访问等级 | A - 提供 YouTube URLs，需自行下载 |
| 获取方式 | <https://www.di.ens.fr/willow/research/howto100m/> |
| 预计空间 | ~TB 级（取决于下载分辨率） |
| 关键模态 | RGB 视频、ASR 文本 |
| 与项目关系 | 人类操作/教学视频的海量预训练数据 |

### Charades / Charades-Ego (2016)

| 维度 | 详情 |
|------|------|
| 规模 | ~10K 视频，157 类日常居家活动；Charades-Ego 为第一人称版本 |
| 标注 | 动作类别、时间边界、多标签 |
| 重要性 | 居家环境人类日常行为标准数据集 |
| 发布年份 | 2016 |
| 访问等级 | A - 直接下载 |
| 获取方式 | <https://allenai.org/plato/charades/> |
| 预计空间 | ~几十 GB |
| 关键模态 | RGB 视频、动作标签 |
| 与项目关系 | 居家场景人-物交互和日常活动 |

### ActivityNet (2015-2017)

| 维度 | 详情 |
|------|------|
| 规模 | 20K 视频，200 类人类活动，约 648 小时 |
| 标注 | 动作类别、时间边界、dense captions |
| 重要性 | 最广泛使用的人类活动视频数据集之一 |
| 发布年份 | 2015-2017 |
| 访问等级 | A - 提供 YouTube URLs |
| 获取方式 | <http://activity-net.org/> |
| 预计空间 | ~百 GB 级 |
| 关键模态 | RGB 视频、caption、时间标注 |
| 与项目关系 | 人类活动视频的通用预训练 |

### YouCook2 (2018)

| 维度 | 详情 |
|------|------|
| 规模 | 2000 烹饪视频，89 菜谱，约 176 小时 |
| 标注 | 流程步骤时间边界、步骤描述、弱监督物体定位 |
| 重要性 | 密集流程标注，天然包含 before/after 状态变化 |
| 发布年份 | 2018 |
| 访问等级 | A - 直接下载 |
| 获取方式 | <http://youcook2.eecs.umich.edu/> |
| 预计空间 | ~几十 GB |
| 关键模态 | RGB 视频、时间边界、步骤文本 |
| 与项目关系 | 流程型人-物交互，before/action/after 状态变化 |

### FineGym (CVPR 2021)

| 维度 | 详情 |
|------|------|
| 规模 | 32K+ 视频，530 细粒度体操动作子类 |
| 标注 | 层级动作标签（事件→组→元素），时间边界 |
| 重要性 | 极细粒度人体运动，是理解复杂人体动态的优质数据 |
| 发布年份 | 2021 (CVPR) |
| 访问等级 | A - 直接下载 |
| 获取方式 | <https://sdolivia.github.io/FineGym/> |
| 预计空间 | ~几十 GB |
| 关键模态 | RGB 视频、细粒度动作层级标签 |
| 与项目关系 | 高精度人体运动建模 |

## P2 级 — 可选补充，特定场景覆盖

### How2Sign (2020)

| 维度 | 详情 |
|------|------|
| 规模 | 80+ 小时连续美式手语 (ASL)，多视角 |
| 标注 | 英语翻译文本、深度图、2D/3D 人体关键点 |
| 重要性 | 极为丰富的手势和身体语言数据，天然的全双工交互模式 |
| 发布年份 | 2020 |
| 访问等级 | A - 直接下载 |
| 获取方式 | <https://how2sign.github.io/> |
| 预计空间 | ~百 GB 级 |
| 关键模态 | 多视角 RGB、深度、2D/3D 姿态、语音、文本 |
| 与项目关系 | 手势/身体语言的全双工交互建模 |

### DFEW (2020)

| 维度 | 详情 |
|------|------|
| 规模 | 16,372 视频片段，来自电影 |
| 标注 | 7 类基本表情（含中性），动态面部表情 |
| 重要性 | 最大规模的 wild 场景动态面部表情数据集之一 |
| 发布年份 | 2020 |
| 访问等级 | A - 直接下载 |
| 获取方式 | <https://aip.seu.edu.cn/> (东南大学) |
| 预计空间 | ~几十 GB |
| 关键模态 | RGB 视频、表情标签 |
| 与项目关系 | 数字人面部表情生成 |

### MAFW (2021)

| 维度 | 详情 |
|------|------|
| 规模 | 10,045 视频片段，来自电影/综艺/短视频 |
| 标注 | 11 维表情分布向量、单标签/多标签表情、双语情感描述文本 |
| 重要性 | 大规模复合情感表达数据集，覆盖多文化场景 |
| 发布年份 | 2021 |
| 访问等级 | A - 直接下载 |
| 获取方式 | <https://mafw-database.github.io/MAFW/> |
| 预计空间 | ~几十 GB |
| 关键模态 | RGB 视频、音频、表情分布、文本描述 |
| 与项目关系 | 多模态情感表达，适合交互场景 |

### IEMOCAP (2008)

| 维度 | 详情 |
|------|------|
| 规模 | 12 小时 dyadic 互动，10 名演员 |
| 标注 | 9 类情感标签、3 维情感维度（valence/activation/dominance）、转录文本 |
| 重要性 | 经典双人自然交互情感数据集，对话+动作+情感 |
| 发布年份 | 2008 |
| 访问等级 | B - 需申请 |
| 获取方式 | <https://sail.usc.edu/iemocap/> |
| 预计空间 | ~几十 GB |
| 关键模态 | 视频、音频、动作捕捉、文本转录 |
| 与项目关系 | 双人交互场景的情感理解 |

### CMU-MOSEI (2018)

| 维度 | 详情 |
|------|------|
| 规模 | 23,454 句子，来自 1,000 个 YouTube 说话者，约 65 小时 |
| 标注 | 6 类情感、5 级情感强度、多模态情感分析 |
| 重要性 | 最大规模的多模态情感分析数据集之一，覆盖丰富的真实说话场景 |
| 发布年份 | 2018 |
| 访问等级 | A - 直接下载 |
| 获取方式 | <http://multicomp.cs.cmu.edu/resources/cmu-mosei-dataset/> |
| 预计空间 | ~几十 GB |
| 关键模态 | 视频、音频、文本 |
| 与项目关系 | 说话者情绪表达，适用于交互场景 |

## 建议优先级

1. **OpenHumanVid** — 邮件申请，专门为视频生成设计，价值最高
2. **EgoVid-5M** — 直接下载，第一人称视频生成专用
3. **Action100M (preview)** — HF 直接下载，可作为大规模预训练
4. **HowTo100M** — YouTube URLs，大规模人类操作场景
5. **FineGym + YouCook2 + Charades** — 直接下载，补充特定场景覆盖
6. **其余 P2 数据集** — 根据后续需求按需补充

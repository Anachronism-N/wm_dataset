# 10_风险与许可

> 风险、许可与发布审计

> 代码许可证、标注/索引许可证、原始音视频权利、模型产物/权重发布权必须分别审计。

| 审计项                | 最低要求            | 阻塞条件            | 记录位置                         |
|--------------------|-----------------|-----------------|------------------------------|
| 代码许可               | 记录仓库版本与LICENSE  |                 | license_ledger               |
| 数据/标注许可            | 保存协议与访问日期       |                 | source级证据账本                  |
| 原始音视频权利            | 明确来源与删除机制       |                 | source_id/takedown映射         |
| 模型权重发布             | 逐来源确认衍生模型条款     |                 | 发布前allowlist                 |
| 风险                 | 概率              | 影响              | 触发信号                         |
| 许可不允许生成训练          | 中               | 极高              | 仅允许理解/benchmark或禁止衍生模型       |
| 预告开放但实际不可下载        | 高               | 高               | will release、仓库空、无文件         |
| 大数据下载拖垮周期          | 高               | 高               | 3天未完成或空间不足                   |
| VLM把意图当结果          | 高               | 极高              | expected与after大量相同           |
| 动作与视频不同步           | 中               | 极高              | 按键/画面延迟或AV offset大           |
| 多人身份混淆             | 高               | 高               | ID switch；speaker/listener错配 |
| 状态变化不可观察           | 高               | 高               | 遮挡/离屏，after不明确               |
| 真实/仿真/机器人域差        | 高               | 中高              | 机械臂/仿真外观污染                   |
| 百小时覆盖不足            | 中               | 中               | 类别集中/组合缺失                    |
| AV联合目标过复杂          | 高               | 高               | loss不收敛/显存暴涨                 |
| 评估泄漏               | 中               | 极高              | 同源/身份/房间跨split               |
| 新Benchmark无release | 高               | 中               | 无法下载/接口不统一                   |

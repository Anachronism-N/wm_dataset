## OpenHumanVid

- **为什么最重要**：CVPR 2025 论文专门为"以人为中心的视频生成"设计的数据集
- **标注特点**：每段视频有 3 类文本描述 — 人物外观(appearance)、动作(motion)、综合(combined)
- **论文效果**：加入该数据集后，生成人像视频的 VBench 各项指标显著提升
- **获取步骤**：
  1. 访问 <https://github.com/fudan-generative-vision/OpenHumanVid>
  2. 按 README 指引填写申请表
  3. 收到邮件后通过链接下载

## EgoVid-5M

- **为什么重要**：NeurIPS 2025 专为"第一人称视频生成"设计
- **标注特点**：包含细粒度动作控制标注（low-level kinematic control），天然适合条件视频生成
- **获取步骤**：直接访问 <https://egovid.github.io/> 下载

## Action100M

- **为什么重要**：Meta 2026.01 发布，147M 动作片段，人类动作覆盖极广
- **标注特点**：分层动作标签 + 时间边界
- **获取步骤**：
  1. HF: `huggingface-cli download facebook/action100m-preview`
  2. 或 GitHub: <https://github.com/facebookresearch/Action100M>
  3. 全量需额外申请

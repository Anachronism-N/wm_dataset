# 不适用于视频生成训练的数据集

> 以下数据集经评估不适合用于 Wan2.2 视频生成训练，建议排除。

---

## 一、纯 3D/动作捕捉数据（无视频）

| 数据集 | 说明 |
|--------|------|
| **AMASS** | 纯 3D 骨骼/网格动作数据库 |
| **Human3.6M** | 主要为 3D 关节姿态数据 |
| **HumanML3D** | 文本-3D 动作向量 |
| **Motion-X** | 3D 全身动作向量 |
| **AIST++** | SMPL 3D 参数为主 |
| **BEAT/BEAT2** | MoCap 动作捕捉数据 |
| **TalkSHOW** | MoCap 动作捕捉数据 |
| **Trinity Speech-Gesture** | MoCap 动作捕捉数据 |
| **CMU Panoptic** | 多视角 3D 动捕 |

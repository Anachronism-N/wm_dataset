## 下载方式说明

| 方式 | 命令 | 需要安装 |
|------|------|---------|
| 直接 wget | `wget -c <URL>` | 系统自带 |
| GitHub | `git clone <URL>` | 系统自带 |
| HuggingFace | `huggingface-cli download --repo-type dataset --local-dir <DIR> <REPO>` | `pip install huggingface_hub` |
| Google Drive | `gdown <URL>` | `pip install gdown` |
| OpenDataLab | `python -c "from opendatalab import OpenDataLab; ..."` | `pip install opendatalab` |

## 执行方式

```bash
# 赋予执行权限
chmod +x dataset/scripts/download_all.sh

# 运行全部下载（部分需要手动步骤）
bash dataset/scripts/download_all.sh

# 或按类别单独运行

# 仅下载数字人小数据集（快速验证）
bash dataset/scripts/download_all.sh  # 修改脚本中的函数调用顺序
```

## 执行优先级建议

1. **第一批（立即执行，可直接下载）**：
   - CREMA-D（GitHub, 几GB）
   - RAVDESS（Zenodo, ~24GB）
   - ViCo（项目页, <20GB）
   - Charades（Allen AI, 数十GB）
   - YouCook2 标注（直链, 几GB）

2. **第二批（需要 HuggingFace 登录）**：
   - InterHuman
   - EgoVid-5M
   - Assembly101 子集
   - EPIC-KITCHENS-100 标注

3. **第三批（需要注册/申请）**：
   - DFEW（注册东南大学）
   - MAFW（注册项目页面）
   - CMU-MOSEI（注册 CMU）
   - EPIC-KITCHENS-100 视频（注册后获取）

|------|-----------|-----------|

# Full-Duplex Multimodal Interactive World Model — 数据文档索引

> 本项目旨在构建一个以人为中心的全双工多模态交互世界模型，由"大脑（VLM/多模态大模型）+ 渲染器（Video Diffusion/DiT）"分层架构组成。当前阶段重点围绕高质量人类中心交互数据构建。

## 数据概览

- **已编目数据集**：~90 个（含 14_遗漏补充候选 15 个）
- **本次调研新增**：12 个高价值遗漏数据集
- **目标训练数据量**：118 小时（首期 2 周交付 40 小时 smoke test）
- **数据类别覆盖**：大规模视频、人脸/人体、对话/数字人、人-物交互、具身/游戏/机器人、评测基准
- **训练阶段**：7 阶段（Stage 0 → Stage 6），从通用视频到长记忆交互
